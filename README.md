# Dev Tools Setup — Cursor IDE + Claude Code + Codex

## Descripción
Este repositorio documenta el proceso de configuración de un entorno de desarrollo
moderno utilizando Cursor IDE junto con las extensiones Claude Code y Codex.
Fue creado como parte del proceso de aplicación a una posición técnica.

---

## Herramientas instaladas

| Herramienta | Descripción | Fuente |
|---|---|---|
| **Cursor IDE** | Editor de código potenciado con IA, basado en VS Code | cursor.com |
| **Claude Code** | Extensión de Anthropic para asistencia de código con IA dentro de Cursor | Marketplace de Cursor |
| **Codex** | Extensión complementaria para generación y autocompletado de código | Marketplace de Cursor |
| **Git** | Sistema de control de versiones para gestionar y subir cambios a GitHub | git-scm.com |

---

## Pasos completados

1. ✅ Descarga e instalación de **Cursor IDE** desde [cursor.com](https://cursor.com)
2. ✅ Instalación de la extensión **Claude Code** desde el Marketplace de Cursor e inicio de sesión
3. ✅ Instalación de la extensión **Codex** desde el Marketplace de Cursor
4. ✅ Creación de un **repositorio público** en GitHub
5. ✅ Clonación del repositorio y apertura en Cursor
6. ✅ Creación de este archivo **README.md**
7. ✅ Commit y push de los cambios a GitHub

---

## Problemas encontrados y soluciones

### Problema 1 — "Git: Clone" no aparecía en Cursor
**Causa:** Git no estaba instalado en el sistema operativo (Windows).  
**Solución:** Se descargó e instaló Git desde [git-scm.com](https://git-scm.com/download/win),
se reinició Cursor y la opción apareció correctamente al volver a buscarla
con `Ctrl + Shift + P`.
